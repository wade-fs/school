# 台灣高中教師 App 框架 (Taiwan High School Teacher App)

這是一個專為台灣高中職教師設計的 Android 應用程式原型，採用 **Jetpack Compose** 與 **Material 3** 開發。本專案以「角色為核心」進行設計，整合了導師、輔導、科任等多種職務的日常需求。

## 🌟 核心功能摘要

### 1. 多角色工作台 (Role-Based Dashboard)
- **輔導教師**: 整合個案管理、加密晤談紀錄、危機事件通報與心情預警系統。
- **導師/科任**: 提供快速點名、班級管理與親師生互動樞紐。

### 2. 專業輔導模組 (Counselor Module)
- **個案管理**: 支援 CSV 學生名冊匯入，提供強大的搜尋與篩選（高風險、休學、法院等標籤）。
- **加密紀錄**: 晤談文字稿使用 AES 加密儲存於本地資料庫，確保學生隱私。
- **語音輸入**: 整合 zh-TW 語音辨識，方便老師在晤談後快速整理紀錄。
- **危機通報**: 內建危機事件記錄表單，自動追蹤嚴重程度與處理進度。
- **心情溫度計**: 班級心情量測與自動預警機制，協助老師及早介入。
- **資源庫**: 預載台灣常用 24 小時求助專線（安心專線、生命線等），支援一鍵撥打。

### 3. 全國學校動態串接
- **教育部資料整合**: 自動從教育部開放平台下載最新（114 學年度）高中職名單。
- **智慧搜尋**: 老師可透過關鍵字快速挑選所屬學校，系統自動帶入校名與官網連結。
- **自動學年切換**: 每年 8/1 自動計算新學年度，並具備資料下載回退 (Fallback) 機制。

### 4. 安全與稽核
- **自動日誌**: 所有敏感資料的存取（檢視紀錄、建立通報等）皆會自動記錄於稽核日誌 (Audit Log)。
- **資料保護**: 本地 Room 資料庫儲存，搭配加密欄位。

## 🚀 技術規格
- **開發語言**: Kotlin 2.0.20
- **UI 框架**: Jetpack Compose (Compose Compiler 2.0.20)
- **資料庫**: Room Persistence Library
- **網路通訊**: Ktor Client (Android Engine)
- **CI/CD**: GitHub Actions (自動編譯檢查與 Release APK 打包)

## 🛠 如何開始
1. 在 Android Studio 開啟此專案。
2. 確保環境已安裝 **JDK 17**。
3. 點擊 **Run** 即可部署至模擬器或實體手機。
4. **CI/CD 設定**: 在 GitHub 設定 `SIGNING_KEY_BASE64`、`KEY_ALIAS` 等 Secrets 後，推送 Tag 即可自動打包 Release APK。

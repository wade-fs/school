# 輔導教師專屬模組規格 (Guidance Counselor Pro)

## 1. 核心邏輯：學生生命週期與狀態管理
- **學生狀態 (Status)**：
  - 在學 (Active)
  - 休學 (Suspended)
  - 轉學 (Transferred)
  - 結案 (Closed)
  - 外部轉介 (Referred): 如心理醫師、社工、法院、少年監獄等。
- **追蹤機制 (Tracking)**：
  - **個案主表**：包含所有基礎資料與「當前法律/醫療狀態」。
  - **歷程記錄 (Timeline)**：類似醫療病歷，包含：
    - 晤談紀錄 (Session Logs): 語音 + 書面。
    - 狀態變更 (Status Changes): 記錄何時休學、何時轉介。
    - 外部追蹤 (External Tracking): 法院開庭進度、監獄訪視紀錄。

## 2. 預約與提醒功能
- **預定晤談 (Appointments)**：
  - 設定下次晤談時間。
  - App 推播提醒教師準備資料。
  - 儀表板顯示「今日預約」與「逾期未晤談」清單。

## 3. 本地儲存架構 (Local-Only Storage)
- **技術選用**：使用 Android **Room Database** 儲存結構化資料。
- **隱私安全性**：所有資料皆儲存在 App 私有目錄，不經過雲端，確保學生個資不外洩。
- **CSV 匯入**：支援從手機本地選擇 CSV 檔案進行學生資料批次更新。

## 4. 專業搜尋介面
- 提供快捷按鈕：顯示「休學中個案」、「高風險個案」、「法院追蹤個案」。
- 姓名與學號混合搜尋。
